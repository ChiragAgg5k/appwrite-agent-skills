# Graphql

The GraphQL API allows you to query and mutate your Appwrite server using GraphQL.

## GraphQL endpoint

`query`
Execute a GraphQL mutation.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `query` | object | Yes | The query or queries to execute. (default: `{}`) |

**Response:** `any`

## GraphQL endpoint

`mutation`
Execute a GraphQL mutation.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `query` | object | Yes | The query or queries to execute. (default: `{}`) |

**Response:** `any`
